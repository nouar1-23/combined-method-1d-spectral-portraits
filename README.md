# Комбинированный метод: Одномерные спектральные портреты (MATLAB)

Данный репозиторий содержит реализацию **комбинированного метода** для построения одномерных спектральных портретов случайных матриц в среде MATLAB.

## Описание метода
Программа реализует алгоритм, который использует спектральный проектор для разделения пространства на инвариантные подпространства. Это позволяет разбить исходную матрицу на блоки, соответствующие собственным значениям внутри и вне окружности заданного радиуса $r$, и эффективно решать уравнение Ляпунова итерационным методом.

**Основные этапы алгоритма:**
1. Вычисление проектора с использованием QR-разложения.
2. Разделение пространства на подпространства через SVD-разложение проектора.
3. Итерационное решение дискретных уравнений Ляпунова для каждого блока (внутри и вне круга) по отдельности.
4. Формирование общего решения $H$ обобщенного уравнения Ляпунова путем сборки (комбинирования) решений, полученных для каждого блока.
5. Вычисление нормы решения $H$ и построение спектрального портрета.
